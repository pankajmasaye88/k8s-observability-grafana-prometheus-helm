K8s Kind Voting App
Setting up a Kubernetes cluster using Kind on an AWS EC2 instance, installing and configuring Grafana and Prometheus.

Overview
This guide covers the steps to:

Launch an AWS EC2 instance.
Install Docker and Kind.
Create a Kubernetes cluster using Kind.
Install and access kubectl.
Install Helm.
Install Grafana and Prometheus using Helm.
Connect and manage your Kubernetes cluster.
